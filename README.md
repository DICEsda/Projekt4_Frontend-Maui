
# Personal Portfolio Website

A modern and responsive developer portfolio built using React, TypeScript, TailwindCSS, and Vite.  
This project showcases my engineering work in software development, IoT, AI, and embedded systems.

---

## Features

- Fast development and build setup using Vite
- Responsive layout for desktop and mobile
- Framer Motion animations for smooth UI transitions
- Project showcase section with visual presentation
- Working contact form using EmailJS
- Clean and modular component structure
- Dark/Light theme support
- Deployed on GitHub Pages

---

| Category | Technologies |
|--------|--------------|

## Getting Started

### Prerequisites
- Node.js and npm installed

### Install dependencies

```bash
npm install
````

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

---

## Deployment

### Deploy to GitHub Pages

```bash
npm install gh-pages --save-dev
npm run build
npm run deploy
```
